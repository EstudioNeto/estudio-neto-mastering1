Palco Mestre Vídeo Player

O reprodutor definitivo para apresentações e música ao vivo

O Palco Mestre é o software de desktop definitivo para cantores e instrumentistas de música ao vivo que não podem se dar ao luxo de parar uma apresentação para encontrar o próximo vídeo de acompanhamento. Agende, mude e controle sua playlist em tempo real, sem interrupções indesejadas no palco.

Sistemas Operacionais Compatíveis:

O Palco Mestre Vídeo Player foi projetado e otimizado para o ambiente Windows nas seguintes versões de 64-bit:

  • Windows 11
  • Windows 10
  • Windows 8 / 8.1
  • Windows 7

Arquitetura de Processamento

O instalador fornecido é compilado para sistemas de 64-bit, que é o padrão estrutural para computadores modernos. Para o correto funcionamento, certifique-se de que seu sistema operacional Windows também utilize essa arquitetura.
